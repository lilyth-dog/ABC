## 문제 해결

### 한글이 깨지는 경우
- Windows: 'Malgun Gothic' 폰트 확인
- Mac: 'AppleGothic' 또는 'NanumGothic' 사용
- Linux: 'NanumGothic' 설치 필요

### 이미지가 너무 큰 경우
- PowerPoint에서 이미지 압축 사용
- 또는 DPI를 200으로 낮춰서 재생성

### 이미지가 보이지 않는 경우
- 파일 경로 확인
- 이미지 뷰어로 직접 열어서 확인
