### 시연 흐름 (2-3분)

#### Step 1: Anima Weaving - 성격 설정 (30초)

**[화면: IdentityConfigurator 컴포넌트]**

**발표자**: 
"먼저 Anima Weaving 단계를 보여드리겠습니다. 사용자가 UI에서 선택과 조정을 하는 동안, 시스템은 실시간으로 행동 신호를 수집합니다."

**[마우스로 옵션 선택하는 모습 보여주기]**

**발표자**: 
"지금 마우스로 옵션을 선택하는데, 시스템은 다음을 측정합니다:
- 의사결정 지연시간: 옵션이 나타난 후 선택까지의 시간
- 마우스 경로 효율성: 최적 경로 대 실제 경로의 비율
- 수정 빈도: 값 변경 횟수"

**[결과 화면 표시]**

**발표자**: 
"이제 실시간으로 계산된 성격 가중치가 표시됩니다. Logic 0.65, Intuition 0.35, Fluidity 0.72, Complexity 0.58입니다. 또한 추론 근거도 함께 보여줍니다: '높은 의사결정 지연시간(3500ms)이 감지되어 Logic 가중치가 높게 계산되었습니다.'"

#### Step 2: 예측 대시보드 (30초)

**[화면: PredictiveInsights 컴포넌트]**

**발표자**: 
"이제 예측 대시보드를 보여드리겠습니다. 사용자가 여러 세션을 진행한 후의 결과입니다."

**[레이더 차트 표시]**

**[영역 차트 표시]**

**발표자**: 
"행동 트렌드는 영역 차트로 현재와 예측을 비교합니다. 각 성격 특성의 변화 방향을 시각화했습니다."

**[스트레스 분석 표시]**

**발표자**: 
"스트레스 분석 결과입니다. 현재 스트레스 레벨은 0.3으로 낮은 수준이며, '경로 효율성 저하'가 감지되었습니다."

#### Step 3: 지속적 학습 효과 (30초)

**[화면: 프로필 진화 타임라인]**

**발표자**: 
"지속적 학습 효과를 보여드리겠습니다. 세션별로 성격 가중치가 어떻게 변화하는지 확인할 수 있습니다."

**[세션 1-3 데이터 표시]**

**발표자**: 
"세션 1-3에서는 빠른 적응이 일어납니다. Logic 가중치가 0.45에서 0.52로 증가했습니다. 신뢰도는 30%에서 50%로 증가했습니다."

**[세션 4-7 데이터 표시]**

**발표자**: 
"세션 4-7에서는 안정화가 일어납니다. 가중치 변동성이 줄어들고, 신뢰도는 70%에서 82%로 증가했습니다."

**[드리프트 감지 알림 표시]**

**발표자**: 
"성격 변화가 감지되면 사용자에게 알림을 보냅니다. 예를 들어, 'Logic 가중치가 15% 증가했습니다'라는 메시지가 표시됩니다."
