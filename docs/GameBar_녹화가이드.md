# Game Bar로 시연 영상 녹화하기
## 간단한 5분 가이드

---

## 빠른 시작 (3단계)

### 1단계: Game Bar 활성화
- `Win + G` 키 누르기
- 처음 사용하면 "게임 바 사용" 활성화

### 2단계: 녹화 시작
- `Win + Alt + R` 키로 녹화 시작/중지
- 또는 Game Bar 창에서 녹화 버튼 클릭

### 3단계: 파일 확인
- 녹화 파일 위치: `C:\Users\[사용자명]\Videos\Captures`
- 파일 형식: `.mp4`

---

## 녹화할 내용 (1-2분)

### Scene 1: Anima Weaving (30초)
1. 브라우저에서 `http://localhost:5173` 열기
2. IdentityConfigurator 화면으로 이동
3. 옵션 선택하는 모습 (마우스 움직임 보이게)
4. 성격 가중치가 계산되는 화면 보여주기
5. 추론 근거 표시 확인

### Scene 2: 예측 대시보드 (30초)
1. PredictiveInsights 화면으로 이동
2. 레이더 차트 보여주기 (30일 후 예측)
3. 영역 차트 보여주기 (행동 트렌드)
4. 스트레스 분석 결과 확인

### Scene 3: 지속적 학습 (30초)
1. 프로필 진화 타임라인 보여주기
2. 세션별 가중치 변화 그래프
3. 신뢰도 증가 표시
4. 드리프트 감지 알림 (있다면)

---

## 녹화 팁

### ✅ DO
- 마우스 움직임을 천천히, 명확하게
- 각 화면에서 2-3초 멈춤 (이해할 시간)
- 핵심 숫자/차트에 마우스 포인터로 강조
- 브라우저 창만 녹화 (불필요한 부분 제외)

### ❌ DON'T
- 너무 빠르게 화면 전환
- 불필요한 대기 시간
- 개인 정보 노출 (이름, 이메일 등)

---

## 녹화 전 체크리스트

- [ ] Backend 실행 확인 (`python api_server.py`)
- [ ] Frontend 실행 확인 (`npm run dev`)
- [ ] 브라우저에서 `http://localhost:5173` 접속 확인
- [ ] 테스트 데이터 준비 (여러 세션 데이터)
- [ ] Game Bar 단축키 확인 (`Win + Alt + R`)

## 녹화 후

### 파일 확인
- 위치: `C:\Users\[사용자명]\Videos\Captures`
- 파일명: `클립_YYYYMMDD_HHMMSS.mp4`

### PowerPoint에 삽입
1. PowerPoint 열기
2. 해당 슬라이드 선택
3. "삽입" → "비디오" → "내 PC의 비디오"
4. 녹화 파일 선택

### 파일 크기
- 1분 영상: 약 20-30MB
- 2분 영상: 약 40-60MB

---

## 문제 해결

### Game Bar가 안 열릴 때
- Windows 설정 → 게임 → 게임 바 → "게임 바 사용" 확인
- `Win + G` 다시 시도

### 녹화가 안 될 때
- 관리자 권한으로 실행
- 다른 녹화 프로그램 종료
- 그래픽 드라이버 업데이트

### 파일이 너무 클 때
- 녹화 시간 줄이기 (1분 이내)
- 해상도 낮추기 (Game Bar 설정)
