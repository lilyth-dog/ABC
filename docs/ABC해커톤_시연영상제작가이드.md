## 화면 녹화 방법

### Windows: OBS Studio

#### 1. 설치
```bash
# OBS Studio 다운로드
https://obsproject.com/
```

#### 2. 설정
1. **소스 추가**: "화면 캡처" 선택
2. **출력 설정**: 
   - 인코더: x264
   - 비트레이트: 2500 Kbps
   - 해상도: 1920x1080
3. **오디오**: 마이크 녹음 (선택)

#### 3. 녹화
1. "녹화 시작" 버튼 클릭
2. 시연 시나리오 진행
3. "녹화 중지" 버튼 클릭
4. 파일 저장 위치 확인

---

### Windows: Game Bar (간단)

#### 1. 활성화
- `Win + G` 키 누르기
- "게임 바 설정" → "게임 바 사용" 활성화

#### 2. 녹화
- `Win + Alt + R` 키로 녹화 시작/중지
- 파일 위치: `C:\Users\[사용자명]\Videos\Captures`

---

### Mac: QuickTime Player

#### 1. 실행
- QuickTime Player 열기
- "파일" → "새 화면 녹화"

#### 2. 녹화
- 녹화 버튼 클릭
- 화면 영역 선택
- 녹화 시작/중지

---
