## 슬라이드 3: 핵심 아이디어

**제목**: 암묵적 행동 신호로 성격 추론

**표**:

| 행동 신호 | 측정 방법 | 성격 특성 |
|----------|----------|----------|
| 🖱️ 마우스 움직임 | 경로 효율성 | 집중력, 안정성 |
| ⏱️ 의사결정 시간 | 선택 지연시간 | 논리적 vs 직관적 |
| 🔄 수정 빈도 | 값 변경 횟수 | 완벽주의, 복잡성 선호 |

**결과**: 4차원 성격 가중치
- Logic (논리성)
- Intuition (직관성)
- Fluidity (유연성)
- Complexity (복잡성)

## 슬라이드 4: AI/ML 파이프라인

**제목**: 머신러닝 시스템 구조

**다이어그램**:
```
[행동 신호 수집] 
    ↓
[특징 추출] → Decision Latency, Revision Rate, Path Efficiency
    ↓
[성격 추론 모델] → 4차원 가중치 예측 (회귀 모델)
    ↓
[온라인 학습] → EMA 알고리즘으로 세션 간 업데이트
    ↓
[예측 모델] → 미래 행동 패턴 예측
    ↓
[이상 감지] → 스트레스 및 비정상 행동 감지
```

**핵심 AI 기법**:
1. 특징 기반 회귀 모델
2. 온라인 학습 (EMA)
3. 시계열 예측
4. 이상 감지
5. 편향 완화

## 슬라이드 5: 핵심 알고리즘

**제목**: 핵심 알고리즘

**1. 성격 추론 모델**
```python
W_L = (latency - 1000) / 4000  # Logic Weight
W_C = (revisions × 0.2) + (latency / 10000)  # Complexity
```
- 심리학 연구 기반 규칙 매핑

**2. 온라인 학습 (EMA)**
```
W_new = α × W_session + (1 - α) × W_history
```
- α = 0.3: 최근 세션 30% 가중치
- 실시간 업데이트, 배치 학습 불필요
- 적응적 학습률

**3. 예측 모델**
- 선형 회귀로 트렌드 예측
- 30일 후 성격 진화 예측
- 스트레스 패턴 자동 분류
